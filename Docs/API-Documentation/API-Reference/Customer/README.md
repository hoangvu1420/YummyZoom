# Customer API Reference

This directory contains comprehensive API documentation for customer-facing endpoints in the YummyZoom platform.

## Available APIs

### Core Customer Features

1. **[Authentication and Profile](./01-Authentication-and-Profile.md)**
   - User authentication via phone OTP
   - Profile management and user settings
   - Account security and preferences

2. **[Restaurant Discovery](./02-Restaurant-Discovery.md)**
   - Search and browse restaurants
   - Menu exploration and filtering
   - Restaurant details and information

3. **[Individual Orders](./03-Individual-Orders.md)**
   - Order placement and management
   - Payment processing integration
   - Order tracking and status updates
   - Order history and receipts

4. **[Reviews and Ratings](./04-Reviews-and-Ratings.md)**
   - Submit restaurant and order reviews
   - Rating system and feedback
   - Review management and moderation

5. **[Pricing Preview](./05-Pricing-Preview.md)**
   - Real-time pricing calculations
   - Coupon validation and discount preview
   - Customization cost calculations
   - Tax and fee breakdowns

### Workflow Guides

- **[Complete Customer Onboarding](./Workflows/01-Complete-Customer-Onboarding.md)**
  - End-to-end customer registration and setup
  - Profile completion and preferences
  - First order experience

- **[TeamCart Collaborative Ordering](./Workflows/02-TeamCart-Collaborative-Ordering.md)**
  - Group ordering and collaboration
  - Shared cart management
  - Payment coordination

- **[Order Tracking and Real-time Updates](./Workflows/03-Order-Tracking-and-Real-time-Updates.md)**
  - Real-time order status updates
  - Delivery tracking integration
  - Notification management

## Getting Started

1. **Authentication**: Start with [Authentication and Profile](./01-Authentication-and-Profile.md) to obtain API access tokens
2. **Discovery**: Use [Restaurant Discovery](./02-Restaurant-Discovery.md) to find restaurants and browse menus
3. **Pricing**: Use the new [Pricing Preview](./05-Pricing-Preview.md) to show accurate pricing before order placement
4. **Ordering**: Complete orders using [Individual Orders](./03-Individual-Orders.md) or [TeamCart](./Workflows/02-TeamCart-Collaborative-Ordering.md)
